# 3行3列の行列の逆行列
標準入力で読み取った行列の逆行列を求めるプログラムです
対応しているのは3×3の行列のみなので、2×2や4×4などの行列には対応していないことに注意してください

##使用方法
'''bash
$ git clone git@github.com:guangjitian755-cell/inverse-matrix.git
$ cd inverse-matrix
$ chmod +x inverse
'''
